Bienvenido a RoutesMirror





Fidelidad Visual

Esta aplicación ha sido desarrollada con un enfoque claro en replicar la apariencia y experiencia de usuario de Amazon Flex. La intención es lograr una fidelidad visual que permita al usuario reconocer de inmediato la interfaz y sentirse familiarizado con su funcionamiento. Cada detalle —desde la disposición de los elementos hasta las transiciones y estilos— ha sido cuidadosamente diseñado para reflejar la estética y lógica de la aplicación original, manteniendo al mismo tiempo una estructura adaptable y optimizada para nuevas funcionalidades.





Funcionalidad

Suministramos una lista aleatoria con paradas reales para que se pueda usar la aplicacion, proximamente agregaremos la capacidad opcional de enlazar una cuenta oficial de amazon para mejorar la experiencia del usuario.
