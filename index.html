Juanidemo

Juanidemo es una aplicacion offline, que se usa en el navegador mediante un servidor lan como xampp. Puede ser usada por las instituciones educativas como medio para realizar las votaciones de personero estudiantil o cualquier otro tipo de votaciones estudiantiles.

La aplicacion me fue solicitada por un grupo de estudiantes que deseaban agilizar los procesos de votacion en su institucion. El servidor fue desarrollado con php y realiza el menor numero de desplazamientos url para evitar la carga de datos innesesaria. Las interfaces y demas sistemas fueron programados con html, css y javascript, todo vanilla. La base de datos fue diseñada en PostGreSQL
